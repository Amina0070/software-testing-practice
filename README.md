# Software Testing Practice Portfolio

## Overview

This repository showcases my software testing and quality assurance skills through manual testing, technical documentation, bug reporting, and test planning using real-world web applications.

As an aspiring cybersecurity professional, I am developing an understanding of software quality, secure software development, and structured testing processes. This project demonstrates my ability to identify issues, document findings clearly, and follow industry-standard testing practices.

---

## Project Objectives

- Perform structured manual testing
- Design and document test cases
- Record and report software defects
- Produce professional QA documentation
- Develop attention to detail and analytical thinking
- Build a portfolio for software testing and cybersecurity roles
