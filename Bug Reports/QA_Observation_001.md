# QA Observation 001

## Title

Login credentials are displayed on the login page.

---

## Type

Usability / Demonstration Environment

---

## Severity

Informational

---

## Description

The login page displays the username and password required to access the application.

This appears to be intentional because the website is designed as a demonstration application for testing and automation practice.

In a production environment, displaying login credentials on the login page would represent a security concern.

---

## Recommendation

For demonstration purposes this behaviour is appropriate.

For production software, credentials should never be publicly displayed and users should authenticate using their own secure accounts.
